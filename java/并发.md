<h1 id ="1">基本概念</h1>

<h2 id ="1.1">同步与异步</h2>

**同步：**需要等待方法调用的返回。

**异步：**当调用方法时候，可以先去执行其他方法，等调用方法执行完毕会通知线程

<h2 id=1.2>并行与并发</h2>

**并行：**多个任务同时运行，只会发生在`多CPU`；

**并发：**多个任务交替执行，在客户看来是并行运行；
